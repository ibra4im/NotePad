# Tkinter Notepad

A simple Notepad application built with **Python** and **Tkinter**. This project demonstrates a functional text editor with essential features such as **New, Open, Save, Save As, Cut, Copy, Paste**, and **keyboard shortcuts**. It is designed for learning **Python GUI development** and **file handling**.

---

## Features

- Multi-line text editing using the `Text` widget  
- Scrollable text area  
- File operations: **New**, **Open**, **Save**, **Save As**  
- Edit operations: **Cut**, **Copy**, **Paste**  
- Keyboard shortcuts:  
  - Ctrl + N → New File  
  - Ctrl + O → Open File  
  - Ctrl + S → Save File  
- About dialog with app information  
- Simple, clean, and responsive interface
